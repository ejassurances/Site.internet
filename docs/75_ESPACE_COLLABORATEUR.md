# 75 - Espace collaborateur

## Objectif

Définir l'espace collaborateur EJ Assurances.

Cet espace doit permettre aux collaborateurs du cabinet de traiter les dossiers qui leur sont attribués, suivre les demandes, gérer les tâches et respecter les obligations internes.

## Fonctions attendues

- tableau de bord ;
- clients attribués ;
- tâches ;
- demandes à traiter ;
- sinistres ;
- emails ;
- rendez-vous ;
- documents ;
- alertes conformité ;
- actions en attente ;
- historique.

## Tableau de bord

Le tableau de bord doit afficher :

- tâches prioritaires ;
- demandes urgentes ;
- rendez-vous à venir ;
- dossiers en retard ;
- pièces manquantes ;
- sinistres suivis ;
- alertes conformité ;
- dernières interactions.

## Droits d'accès

Le collaborateur doit voir :

- ses clients attribués ;
- les dossiers affectés ;
- les documents nécessaires à son traitement ;
- les emails rattachés aux dossiers autorisés ;
- les tâches qui lui sont assignées.

Il ne doit pas voir :

- les dossiers non attribués sans autorisation ;
- les paramètres cabinet sensibles ;
- les données financières ou conformité hors périmètre.

## Workflows

Workflows concernés :

- traitement d'une demande ;
- suivi d'un client ;
- relance ;
- préparation rendez-vous ;
- suivi sinistre ;
- demande de pièce ;
- transmission au courtier admin ;
- clôture de tâche.

## Points ouverts

- niveaux de droits collaborateur ;
- périmètre exact par mission ;
- règles de délégation ;
- indicateurs de productivité ;
- validations nécessaires.
