# Espaces connectés

## Objectif du fichier
Décrire les espaces connectés et les services disponibles pour chaque profil utilisateur.

## Décisions validées
- Proposer un espace client pour gérer son dossier, ses contrats et ses demandes.
- Offrir un espace courtier pour piloter les dossiers clients et accéder aux outils commerciaux.
- Fournir un espace mandataire pour suivre son portefeuille et ses commissions.
- Concevoir un espace prescripteur pour suivre les recommandations et les leads.
- Prévoir un espace administration pour la gestion globale et les rapports.

- Les interfaces de tableau de bord et les indicateurs clés.
- Les modalités de validation des accès.
- Les options d’intégration avec des systèmes externes.
