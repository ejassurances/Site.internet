# Design system

## Objectif du fichier
Documenter les principes visuels et les composants du design system pour la plateforme.

## Décisions validées
- Concevoir une identité visuelle professionnelle, chaleureuse et rassurante.
- Prévoir des composants réutilisables pour les boutons, cartes, formulaires et layouts.
- Structurer les styles pour être cohérents sur le site public, le centre de connaissances et les espaces connectés.
- Inclure des règles d’accessibilité et de lisibilité.

## Éléments à préciser plus tard
- La palette de couleurs définitive.
- La typographie, les icônes et les styles de boutons.
- Les composants prioritaires à développer en premier.
- Le guide d’UX pour les interfaces mobiles et desktop.
