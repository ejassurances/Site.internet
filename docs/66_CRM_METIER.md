# 66 - CRM métier EJ Assurances

## Objectif

Définir le CRM métier interne d'EJ Assurances.

Le CRM doit devenir l'outil central du cabinet pour piloter la relation client, les opportunités, les contrats, la conformité, les sinistres et l'activité commerciale.

## Périmètre fonctionnel

Le CRM doit couvrir :

- gestion des prospects ;
- gestion des clients ;
- gestion des contrats ;
- gestion des projets ;
- gestion des documents ;
- gestion des rendez-vous ;
- gestion des tâches ;
- gestion des emails ;
- gestion des demandes clients ;
- gestion des sinistres ;
- gestion des réclamations ;
- suivi commercial ;
- suivi conformité ;
- suivi mandataires ;
- suivi prescripteurs ;
- suivi collaborateurs ;
- historique complet de la relation client ;
- journal d'audit.

## Rôles concernés

- courtier admin ;
- collaborateurs ;
- mandataires ;
- prescripteurs ;
- clients ;
- prospects ;
- partenaires ;
- administrateur technique.

## Objets métier principaux

- prospect ;
- client ;
- contrat ;
- projet ;
- demande ;
- sinistre ;
- réclamation ;
- document ;
- email ;
- rendez-vous ;
- tâche ;
- interaction ;
- consentement ;
- devoir de conseil ;
- audit log.

## Principes clés

- une fiche client doit centraliser toute la relation ;
- chaque action importante doit être historisée ;
- chaque document doit être rattaché à un dossier ;
- chaque demande doit avoir un statut, un responsable et un délai ;
- les droits d'accès doivent être strictement limités au rôle de l'utilisateur ;
- les obligations conformité doivent être intégrées au parcours et non traitées après coup.

## Fiche client cible

La fiche client doit agréger :

- identité ;
- coordonnées ;
- situation familiale ;
- besoins exprimés ;
- contrats ;
- projets ;
- documents ;
- emails ;
- rendez-vous ;
- tâches ;
- sinistres ;
- réclamations ;
- mandataire ou prescripteur associé ;
- historique complet ;
- consentements ;
- classeur ACPR ;
- journal d'audit.

## Tableaux de bord

Le CRM doit proposer des vues :

- cabinet ;
- commercial ;
- production ;
- conformité ;
- sinistres ;
- mandataires ;
- prescripteurs ;
- collaborateurs.

## Points de vigilance

- ne pas mélanger CRM métier et simple base de contacts ;
- ne pas automatiser une décision réglementée sans validation humaine ;
- conserver les preuves de conseil ;
- garantir la confidentialité des situations familiales sensibles ;
- prévoir une traçabilité complète.
