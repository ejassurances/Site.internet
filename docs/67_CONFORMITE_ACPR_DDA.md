# 67 - Conformité ACPR et DDA

## Objectif

Définir les exigences conformité liées à l'activité de courtage en assurances.

Le CRM doit aider le courtier à respecter ses obligations ACPR et DDA sans automatiser une décision réglementée sans validation humaine.

## Exigences principales

Le CRM doit prévoir :

- document d'entrée en relation ;
- recueil des besoins ;
- devoir de conseil ;
- recommandation personnalisée ;
- traçabilité des échanges ;
- conservation des preuves ;
- gestion des réclamations ;
- gestion des conflits d'intérêts ;
- informations précontractuelles ;
- justificatifs clients ;
- contrôle des mandataires ;
- audit interne ;
- journalisation des actions ;
- conformité DDA ;
- exigences ACPR.

## Documents réglementaires

Documents à gérer :

- DER ;
- recueil des besoins ;
- devoir de conseil ;
- recommandation ;
- documents précontractuels ;
- mandats ;
- consentements ;
- justificatifs ;
- preuves de remise ;
- réclamations ;
- courriers sensibles.

## Recueil des besoins

Le recueil doit permettre d'identifier :

- situation familiale ;
- situation professionnelle ;
- situation financière ;
- contrats existants ;
- objectifs ;
- besoins exprimés ;
- risques identifiés ;
- contraintes ;
- préférences client ;
- niveau de compréhension.

## Devoir de conseil

Le module doit documenter :

- informations recueillies ;
- solutions étudiées ;
- solution recommandée ;
- justification ;
- alternatives ;
- exclusions ;
- limites ;
- validation humaine ;
- preuve de remise au client.

## Règles de validation

- l'IA peut préparer un brouillon ;
- le courtier habilité valide ;
- aucune recommandation ne doit être remise sans contrôle humain ;
- chaque version doit être archivée ;
- chaque remise doit être tracée.

## Audit et preuves

Le CRM doit conserver :

- date et auteur des actions ;
- documents remis ;
- échanges importants ;
- décisions ;
- validations ;
- modifications ;
- consentements ;
- refus ou arbitrages client.

## Points ouverts

- modèle final du DER ;
- durée de conservation ;
- format de preuve de remise ;
- matrice de contrôle mandataires ;
- procédure de réclamation détaillée.
