# 76 - Espace prospect

## Objectif

Définir l'espace prospect EJ Assurances.

L'espace prospect doit accompagner une personne avant sa transformation en client, tout en collectant les consentements nécessaires.

## Fonctions attendues

- création de compte prospect ;
- diagnostic protection familiale ;
- dépôt de pièces ;
- suivi de demande ;
- rendez-vous ;
- messagerie ;
- transformation en client ;
- consentements RGPD ;
- informations précontractuelles.

## Parcours prospect

Étapes :

1. demande initiale ;
2. création du compte ;
3. acceptation des consentements ;
4. accès au diagnostic ;
5. dépôt éventuel de pièces ;
6. prise de rendez-vous ;
7. qualification par le cabinet ;
8. transformation en client.

## Données collectées

- identité ;
- email ;
- téléphone ;
- situation familiale ;
- besoin principal ;
- urgence ;
- personnes à protéger ;
- documents transmis ;
- consentements ;
- historique des échanges.

## Transformation en client

La transformation doit :

- conserver l'historique ;
- conserver les consentements ;
- rattacher les documents ;
- créer une fiche client ;
- initialiser le dossier conformité ;
- créer les premières tâches cabinet.

## Points ouverts

- distinction technique entre prospect et client ;
- durée de conservation des prospects non transformés ;
- niveau d'accès documentaire ;
- modèle de consentement ;
- contenu du premier email de connexion.
